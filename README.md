# R-Ladies Rome Workshop - Time Series Forecasting with Linear Regression

The workshop is beginner-friendly, and it covers the following topics:

- How to frame time series as a regression problem
- Feature engineering techniques for time-aware data
- Modeling trend, seasonality, outliers, and breaks
- Practical tips for evaluation and validation



Workshop recording: https://www.youtube.com/watch?v=lk3a3GQ7kc8
